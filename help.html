1. Introduzione

Benvenuto in DICOM Import & Edit, uno strumento avanzato per la gestione, modifica e invio dei file DICOM. Questo manuale ti guiderà attraverso tutte le funzionalità dell'applicazione, permettendoti di sfruttare al massimo il suo potenziale.

2. Panoramica delle funzionalità principali

  • Importazione di file DICOM da diverse fonti
  • Modifica dell'ID Paziente nei file DICOM
  • Invio dei file DICOM al server PACS
  • Gestione delle configurazioni del server PACS
  • Supporto per operazioni batch e gestione della coda di file

3. Installazione e Configurazione

Requisiti di sistema

  • Sistema operativo: Windows 10 o successivo
  • .NET Framework 8.0
  • Spazio su disco: minimo 100 MB
  • RAM: minimo 4 GB

Procedura di installazione

  1. Scarica l'installer di DICOM Import & Edit.
  2. Esegui il file .exe e segui le istruzioni sullo schermo.
  3. L'installer verificherà la presenza di .NET Framework 8.0 e, se necessario, lo scaricherà e lo installerà automaticamente.
  4. Una volta completata l'installazione, avvia DICOM Import & Edit dal menu Start o dall'icona sul desktop.

Configurazione iniziale

  1. All'avvio dell'applicazione, accedi al menu "Impostazioni" per configurare i parametri del server PACS.
  2. Inserisci l'indirizzo IP del server, il numero di porta, l'AETitle locale e remoto.
  3. Salva le impostazioni e chiudi la finestra di configurazione.
  4. Al primo avvio, è normale che il programma avvisi che non esiste un file di configurazione e carichi i parametri di default.

4. Utilizzo dell'Applicazione

Importazione dei file DICOM

  1. Clicca su "File" -> "Importa" e seleziona la fonte dei file DICOM (cartella, DICOMDIR, singolo file).
  2. Seleziona i file o la cartella da importare e clicca su "Apri".
  3. I file importati verranno visualizzati nella tabella principale.

Modifica dell'ID Paziente

  1. Seleziona uno o più file dalla tabella.
  2. Inserisci il nuovo ID Paziente nel campo dedicato.
  3. Clicca su "Modifica ID Paziente" per applicare le modifiche.

Invio dei file al PACS

  1. Seleziona i file da inviare dalla tabella principale.
  2. Clicca su "Invia al PACS".
  3. Monitora lo stato dell'invio tramite la barra di progresso e le notifiche.

Gestione delle configurazioni

  1. Accedi al menu "Impostazioni".
  2. Modifica i parametri di connessione al server PACS.
  3. Salva le impostazioni e chiudi la finestra.

Test C-ECHO

  1. Nella finestra delle impostazioni, clicca su "Test C-ECHO".
  2. Verifica che la connessione al server PACS sia funzionante.

5. Risoluzione dei Problemi

Errori comuni e soluzioni

  • Errore: "Per favore, seleziona almeno un esame dalla tabella."
    Assicurati di aver selezionato almeno un esame dalla tabella prima di procedere con l'operazione.

6. Domande Frequenti (FAQ)

  1. Perché ricevo un errore quando provo a modificare l'ID Paziente?
    Assicurati che l'ID Paziente inserito non sia uguale a quello attuale.
    Verifica che tutti i file selezionati appartengano allo stesso paziente.
  2. Come posso verificare la connessione al server PACS?
    Utilizza la funzione di test C-ECHO nelle impostazioni.
  3. Posso importare esami di pazienti diversi contemporaneamente?
    Sì, ma assicurati di non selezionare file di pazienti diversi quando modifichi l'ID Paziente.

8. Aggiornamenti e Versioni

Tieniti aggiornato sulle ultime versioni di DICOM Import & Edit per usufruire delle nuove funzionalità e dei miglioramenti.

Come aggiornare

  1. Visita il sito ufficiale o il repository del progetto per scaricare la versione più recente dell'installer.
  2. Esegui l'installer e segui le istruzioni sullo schermo. Il programma di installazione aggiornerà automaticamente la versione esistente senza perdere le configurazioni salvate.
