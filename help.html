Guida utente - DICOM Import & Edit

1. Introduzione

Benvenuto nella guida utente di DICOM Import & Edit. Questa applicazione è stata progettata per facilitare l'importazione, la modifica e l'invio di file DICOM. Seguendo questa guida, sarai in grado di utilizzare tutte le funzionalità del programma.

2. Installazione

Per installare DICOM Import & Edit, eseguire il file di installazione e seguire le istruzioni sullo schermo.

3. Configurazione delle impostazioni

Per accedere alle impostazioni, cliccare sull'icona dell'ingranaggio in basso a destra nella finestra principale. Si aprirà il form delle impostazioni dove è possibile configurare i parametri del server PACS.

3.1 Parametri Server PACS

Nella sezione dei parametri del server PACS, è possibile configurare i seguenti campi:

  • AE Title Locale: Il titolo AE (Application Entity) per il dispositivo locale.
  • AE Title Server: Il titolo AE per il server PACS.
  • Indirizzo IP Server: L'indirizzo IP del server PACS.
  • Porta: La porta utilizzata per comunicare con il server PACS.
  • Timeout [ms]: Il tempo di timeout in millisecondi per le comunicazioni con il server PACS.

3.2 Esecuzione del test C-ECHO

È possibile eseguire un test C-ECHO per verificare la connettività con il server PACS. Per fare ciò, cliccare sul pulsante "Esegui C-ECHO". Verrà visualizzato un messaggio di conferma se il test ha esito positivo.

3.3 Validazione degli input

Il form delle impostazioni include un sistema di validazione degli input che verifica se i parametri sono valori che rientrano nei range corretti e sono formattati correttamente. Se un valore inserito non è valido, il campo diventerà rosso fino a quando il parametro non è corretto. Una volta corretto, il campo diventerà verde.

4. Interfaccia principale

L'interfaccia principale di DICOM Import & Edit è composta da vari pulsanti e tabelle che consentono di importare e gestire i file DICOM.

5. Importazione di file DICOM

È possibile importare file DICOM utilizzando uno dei tre pulsanti nella parte superiore dell'interfaccia principale:

  • Apri file DICOM: consente di selezionare e importare singoli file DICOM.
  • Apri DICOMDIR: consente di selezionare e importare una directory DICOMDIR.
  • Apri cartella: consente di selezionare e importare una cartella contenente file DICOM.

6. Visualizzazione degli esami importati

Dopo aver importato i file DICOM, questi saranno visualizzati nella tabella principale. È possibile selezionare un esame per visualizzarne i dettagli.

7. Modifica dell'ID Paziente

Per modificare l'ID Paziente, inserire il nuovo ID nel campo di testo e cliccare su "Modifica ID Paziente".

8. Invio al PACS

Per inviare i file DICOM al PACS, cliccare sul pulsante "Invia al PACS".
