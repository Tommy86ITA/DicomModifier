 \hypertarget{md__user_guide_autotoc_md1}{}\doxysection{\texorpdfstring{Panoramica dell\textquotesingle{}Interfaccia Utente}{Panoramica dell\textquotesingle{}Interfaccia Utente}}\label{md__user_guide_autotoc_md1}
L\textquotesingle{}interfaccia utente di DICOM Modifier è progettata per essere intuitiva e facile da usare. Di seguito viene fornita una descrizione delle principali sezioni dell\textquotesingle{}interfaccia e delle loro funzionalità.




\begin{DoxyEnumerate}
\item {\bfseries{Selezione file}}\+: Questa sezione consente di importare file DICOM, DICOMDIR o intere cartelle contenenti file DICOM.
\begin{DoxyItemize}
\item {\bfseries{Apri file DICOM}}\+: Consente di selezionare e importare un singolo file DICOM.
\item {\bfseries{Apri file DICOMDIR}}\+: Consente di selezionare e importare un file DICOMDIR.
\item {\bfseries{Apri cartella}}\+: Consente di selezionare e importare tutti i file DICOM contenuti in una cartella.
\end{DoxyItemize}
\item {\bfseries{Tabella degli esami}}\+: Visualizza le informazioni sugli esami DICOM importati, tra cui\+:
\begin{DoxyItemize}
\item Cognome e nome del paziente
\item Data di nascita
\item ID Paziente
\item Descrizione dello studio
\item Data dell\textquotesingle{}esame
\item Modalità
\item Serie
\item Immagini
\end{DoxyItemize}
\item {\bfseries{Nuovo ID Paziente (opzionale)}}\+: Questa sezione permette di inserire un nuovo ID paziente per i file selezionati nella tabella.
\begin{DoxyItemize}
\item {\bfseries{Inserire qui il nuovo ID}}\+: Campo di testo per inserire il nuovo ID paziente.
\item {\bfseries{Modifica ID Paziente}}\+: Pulsante per applicare il nuovo ID paziente ai file selezionati.
\end{DoxyItemize}
\item {\bfseries{Invio}}\+: Questa sezione consente di inviare i file DICOM selezionati al server PACS.
\begin{DoxyItemize}
\item {\bfseries{Invia al PACS}}\+: Pulsante per avviare l\textquotesingle{}invio dei file DICOM al server PACS.
\end{DoxyItemize}
\item {\bfseries{Pulisci}}\+: Questo pulsante permette di cancellare la coda dei file DICOM.
\item {\bfseries{Menu delle impostazioni}}\+: Contiene opzioni aggiuntive e informazioni sul programma.
\begin{DoxyItemize}
\item {\bfseries{Esci}}\+: Chiude il programma.
\item {\bfseries{About}}\+: Mostra le informazioni sul programma e sullo sviluppatore.
\item {\bfseries{Impostazioni}}\+: Apre la finestra delle impostazioni per configurare i parametri del server PACS.
\end{DoxyItemize}
\item {\bfseries{Barra di stato}}\+: Mostra lo stato corrente del programma e il numero di file importati.
\end{DoxyEnumerate}\hypertarget{md__user_guide_autotoc_md2}{}\doxysection{\texorpdfstring{Come Utilizzare DICOM Modifier}{Come Utilizzare DICOM Modifier}}\label{md__user_guide_autotoc_md2}

\begin{DoxyEnumerate}
\item {\bfseries{Importazione dei file DICOM}}
\begin{DoxyItemize}
\item Clicca su "{}\+Apri file DICOM"{} per importare un singolo file DICOM.
\item Clicca su "{}\+Apri file DICOMDIR"{} per importare un file DICOMDIR.
\item Clicca su "{}\+Apri cartella"{} per importare tutti i file DICOM contenuti in una cartella.
\end{DoxyItemize}
\item {\bfseries{Modifica dell\textquotesingle{}ID Paziente}}
\begin{DoxyItemize}
\item Seleziona uno o più esami nella tabella degli esami.
\item Inserisci il nuovo ID paziente nel campo di testo "{}\+Inserire qui il nuovo ID"{}.
\item Clicca su "{}\+Modifica ID Paziente"{} per applicare il nuovo ID paziente ai file selezionati.
\end{DoxyItemize}
\item {\bfseries{Invio dei file DICOM al server PACS}}
\begin{DoxyItemize}
\item Seleziona uno o più esami nella tabella degli esami.
\item Clicca su "{}\+Invia al PACS"{} per avviare l\textquotesingle{}invio dei file DICOM al server PACS.
\end{DoxyItemize}
\item {\bfseries{Impostazioni del programma}}
\begin{DoxyItemize}
\item Clicca su "{}\+Impostazioni"{} nel menu delle impostazioni per aprire la finestra delle impostazioni.
\item Configura i parametri del server PACS (IP, porta, AE Title, timeout, AE Title locale).
\item Salva le impostazioni cliccando su "{}\+Salva"{}.
\end{DoxyItemize}
\item {\bfseries{Informazioni sul programma}}
\begin{DoxyItemize}
\item Clicca su "{}\+About"{} nel menu delle impostazioni per visualizzare le informazioni sul programma e sullo sviluppatore.
\end{DoxyItemize}
\item {\bfseries{Uscita dal programma}}
\begin{DoxyItemize}
\item Clicca su "{}\+Esci"{} nel menu delle impostazioni per chiudere il programma.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{md__user_guide_autotoc_md3}{}\doxysection{\texorpdfstring{Note}{Note}}\label{md__user_guide_autotoc_md3}

\begin{DoxyItemize}
\item Assicurati di configurare correttamente i parametri del server PACS nelle impostazioni prima di inviare i file.
\item Dopo aver modificato l\textquotesingle{}ID paziente, verifica che i file siano stati aggiornati correttamente nella tabella degli esami.
\end{DoxyItemize}

Questa guida copre le funzionalità principali di DICOM Modifier e dovrebbe aiutarti a utilizzare il programma in modo efficace. Se hai ulteriori domande, non esitare a contattare il supporto tecnico.
